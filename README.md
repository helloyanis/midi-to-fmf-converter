# MIDI to FMF (Flipper Music Format Converter)

Convert MIDI files! Play them on your flipper!

## How to use

- Install the [Music Player](https://lab.flipper.net/apps/music_player) app on your Flipper
- [Go on the website](https://midi2fmf.🦊💻.ws)
- Pick a MIDI file. The flipper zero can only play a single note at a time. So it's best to pick a simple MIDI file without many simultaneous notes. You can find plenty of MIDI files for popular songs on the [Online Sequencer website](https://onlinesequencer.net/sequences)
- Pick a MIDI track (the equivalent of an instrument in the midi file). You can preview the track by using the website.
- Download the .fmf track, place it on yout flipper in `/apps_data/music_player`
- On your flipper, open the file and enjoy!
